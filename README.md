\# 📊 AI Trading Risk Analyzer



An AI-based system that analyzes trader behavior using trading data and market sentiment to predict risk levels and provide actionable recommendations.



\---



\## 🚀 Project Overview



This project focuses on understanding trader behavior in financial markets. It uses machine learning to identify risky trading patterns such as overtrading and excessive losses. The system predicts risk levels and classifies traders into different behavioral types.



\---



\## 🎯 Key Features



\- Predicts trading risk level (High / Medium / Low)

\- Uses Random Forest for accurate prediction

\- Identifies trader type using K-Means clustering

\- Incorporates market sentiment (Fear / Neutral / Greed)

\- Provides actionable trading recommendations

\- Interactive web application using Streamlit



\---

## 🧠 Working Process

1. User enters trading data (PnL)
2. System generates or fetches market sentiment
3. Machine learning model predicts risk level
4. K-Means clustering identifies trader type
5. System displays results and recommendations
